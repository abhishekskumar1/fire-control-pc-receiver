# Fire Control PC Receiver

Fire Control PC Receiver is the Windows companion app for Fire Control.

It receives commands from the paired Android controller through Bluetooth Serial COM port and performs local keyboard and mouse actions on the user’s own PC.

Created by Abhishek S Kumar / WILDCODE Studios.

## Important

This repository contains only the Windows PC receiver side of Fire Control.

The Android controller app, brand assets, logos, icons, and official product branding are not included in this open-source release.

## Features

- Bluetooth COM port connection
- Keyboard control
- Mouse control
- Touchpad movement
- Scroll support
- Shortcut command support
- Tray background mode
- Optional start with Windows

## Privacy

Fire Control PC Receiver does not require internet access for remote control.

It does not collect, upload, or sell user data.
